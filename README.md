## Запуск проекта
npm install  
npm run dev 

---

## Возможности
- Добавить ингредиент в конструктор бургера.  
- Перемещать ингредиенты вверх/вниз.  
- Удалять ингредиенты.  
- Создавать заказ.  
- Просматривать ленту заказов.  
- Управлять профилем пользователя.

--- 

## Стек
- React 18 (Functional Components, Hooks).  
- TypeScript.  
- Vite.  
- Redux Toolkit + React-Redux.  
- Tailwind CSS.
